Definición - Angular

Angular es una plataforma para la construcción de aplicaciones web de computadoras de escritorio y aplicaciones nativas de celulares. Es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.
