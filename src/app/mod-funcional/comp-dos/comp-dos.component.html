Componentes

Es el elemento en el cual estará la codificación de la aplicación, es decir, la parte visual con las etiquetas html y la parte lógica con el uso de typescript. Estos componentes están alojados en la cajita que denominamos modulo.

Directivas

Básicamente son como variables condicionales que pueden ser declaradas y usadas como elemento para interactuar con nuestras etiquetas del html.
